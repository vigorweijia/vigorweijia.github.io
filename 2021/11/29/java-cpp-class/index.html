Java中使用JNI调用C++ class

Java中调用C++ class

一般使用Java自带的jni调用C++的函数，而无法直接调用C++的类，在javac -h中自动生成的头文件中，可以看到extern “C”关键字，表示告诉编译器以C语言的方式编译以下代码，也就是说并不支持C++类的操作。

如果使用jni调用C++类，需要间接调用。C++的对象有一个固定的地址，这个地址相当于指向这个类的指针，那么可以在Java中封装一层，使用指针来操作这个C++类。
